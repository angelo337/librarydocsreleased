Convenience class for implementing DocumentTransforms. Implementations only need to implement DocumentTransform.transform(java.io.ByteArrayOutputStream, java.io.OutputStream, com.google.enterprise.adaptor.Metadata, java.util.Map), although they should also likely have a static factory method as defined in DocumentTransform.

isRequired

public boolean isRequired()
Description copied from interface: DocumentTransform
If this property is true, a failure of this transform will cause the entire transform pipeline to abort. This is useful in the case where a particular transform is required in order to server data. For example, a transform tasked with redacting or filtering document content. If this is false and a error occurs, this transform is treated as a identity transform.
